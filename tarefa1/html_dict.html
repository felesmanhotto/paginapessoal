Dicionário Tags HTML

  • !DOCTYPE html: Define o tipo de documento como HTML5.
  • html: Define o início do documento HTML.
  • head: Contém metadados e referências externas para o documento HTML, como o título da página, metadados de idioma, charset e links para folhas de estilo externas.
  • body: Contém o corpo de um documento HTML.
  • meta: Define qualquer informação de metadados que não podem ser definidos por outros elementos HTML.
  • title: Define o título da página exibido na barra de título do navegador.
  • link: Especifica as relações entre o documento atual e um recurso externo.
  • h1: Define o cabeçalho de nível 1, geralmente utilizado para títulos principais.
  • h2: Define um cabeçalho de nível 2, geralmente utilizado para subtítulos.
  • h3: Define um cabeçalho de nível 3, geralmente utilizado para títulos de subseções.
  • p: Define um parágrafo no documento HTML.
  • strong: Define um texto em negrito.
  • nav: Define uma seção de navegação no documento HTML.
  • ul: Define uma lista não ordenada de itens.
  • a: Define um link no documento HTML.
  • div: Define uma divisão do conteúdo.
  • section Define uma seção do documento HTML.
  • footer: Define um rodapé no documento HTML.
  • img: Define uma imagem no documento HTML.
  • address: Define informações de contato.
